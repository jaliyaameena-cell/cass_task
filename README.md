# IEEE Circuits and Systems Society (CASS) – Kerala Chapter Website

## Overview

This project is a **responsive website prototype** designed for the **IEEE Circuits and Systems Society (CASS) Kerala Chapter**.
The website provides information about the chapter, upcoming events, past activities, leadership team, and contact details.

The goal of this project is to create a **modern, clean, and professional web platform** that represents the IEEE CASS Kerala Chapter and helps students, researchers, and professionals stay informed about activities and opportunities.

---

## Features

### 1. Home Page

* Hero section with chapter title and introduction
* Brief overview of the IEEE Circuits and Systems Society
* Call-to-action buttons for navigation

### 2. About Section

* Introduction to IEEE CASS
* Vision and mission of the Kerala Chapter
* Information about the role of circuits and systems in modern technology

### 3. Events Section

* Displays **upcoming events**
* Shows **past events**
* Event cards include title, date, location, and short description

### 4. Team Section

* Displays the **executive committee members**
* Each member includes:

  * Name
  * Role
  * Photo placeholder

### 5. Statistics / Achievements

* Highlights chapter achievements such as:

  * Number of members
  * Events conducted
  * Workshops organized

### 6. Contact Section

* Contact form for inquiries
* Email contact
* Social media links

## Technologies Used

This project was built using modern web technologies:

* **React** – Frontend framework for building the user interface
* **Vite** – Fast development and build tool
* **TypeScript** – Typed JavaScript for better code reliability
* **Tailwind CSS** – Utility-first CSS framework for responsive design
* **shadcn/ui** – UI components for modern interface design

## Installation

Clone the repository:

```
git clone <your-repository-link>
```

Navigate to the project directory:

```
cd kerala-cass-connect
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open your browser and go to:

```
http://localhost:5173
```
